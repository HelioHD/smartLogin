## Rodando Localmente

Clone o Projeto ou Baixe

```bash
  git clone 
```

Vá para o diretorio do projeto 

```bash
  cd my-project
```

Instale as dependencias

```bash
  npm install
```

Inicie o Servidor

```bash
  npm start
```


Para reiniciar o Servidor

```bash
  rs
```
